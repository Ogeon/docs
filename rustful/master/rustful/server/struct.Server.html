Used to build and run a server.

Each field has a corresponding modifier method and calls to these methods can be chained for quick setup.

let server_result = Server::new().port(8080).handlers(router).run();

match server_result {
    Ok(_server) => {},
    Err(e) => println!("could not start server: {}", e.description())
}

Fields

handlers

One or several response handlers.

host

The host address and port where the server will listen for requests.

protocol

The type of HTTP protocol.

threads

The number of threads to be used in the server task pool. The default (None) is based on recommendations from the system.

server

The content of the server header.

content_type

The default media type.

log

Tool for printing to a log.

context_plugins

The context plugin stack.

response_plugins

The response plugin stack.
