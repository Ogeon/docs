Linear HSV color space with an alpha component.

HSV is a cylindrical version of RGB, where the value component determines the brightness of the color, and not the lightness. The difference is that, for example, red (100% R, 0% G, 0% B) and white (100% R, 100% G, 100% B) has the same brightness (or value), but not the same lightness.
