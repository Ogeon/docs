Struct palette::Alpha [−] [src]

pub struct Alpha<C, T: Float> {
    pub color: C,
    pub alpha: T,
}

An alpha component wrapper for colors.

Fields

color

The color.

alpha

The transparency component. 0.0 is fully transparent and 1.0 is fully opaque.
