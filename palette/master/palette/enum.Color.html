A generic color type.

The Color may belong to any color space and it may change depending on which operation is performed.

Variants

Luma

Linear luminance.

Rgb

Linear RGB.

Xyz

CIE 1931 XYZ.

Lab

CIE L*a*b* (CIELAB).

Lch

CIE L*C*h°, a polar version of CIE L*a*b*.

Hsv

Linear HSV, a cylindrical version of RGB.

Hsl

Linear HSL, a cylindrical version of RGB.
